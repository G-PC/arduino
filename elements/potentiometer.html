🎚️ Потенциометр

🔌 Схема подключения

🎚️ Подключение: левый вывод → 5V, средний → A0, правый → GND.

💻 Код для Arduino IDE

// Потенциометр на A0 управляет яркостью LED на пине 9
int potPin = A0;
int ledPin = 9;

void setup() {
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int potValue = analogRead(potPin);
  int brightness = map(potValue, 0, 1023, 0, 255);
  analogWrite(ledPin, brightness);
  
  Serial.print("Значение потенциометра: ");
  Serial.println(potValue);
  
  delay(50);
}

✨ Вращайте потенциометр — яркость светодиода плавно меняется!
